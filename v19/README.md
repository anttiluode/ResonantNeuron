This folder builds the **medial septum as a Kuramoto oscillator bank**
(Vardalakis et al. 2024), generating the theta rhythm that paces the resonator
neuron — the clock that v16 (latency = logic depth), v17 (linear carry

with the loop closed back to the original accident.

## The short version

The resonator neuron's own results kept demanding a clock. The brain's clock is
theta, generated by the septum — a population of coupled oscillators poised at
the edge of synchrony, which is structurally the same edge-of-chaos mechanism
as the Homeostatic Coupler that produced the **original accident**. This folder

## The files

**`septum.py`** — the Kuramoto septum. N coupled phase oscillators generating a
theta rhythm (verified 7.00 Hz, coherence 0.82) with a phase-reset term. Run it
to see theta self-organize and a pulse reset its phase.

**`septum_paced_adder.py`** — the loop closed. Replaces the hand-coded
scheduler with the generated theta clock: each theta peak gates one bit-stage,
the carry ripples one theta cycle at a time, and stage completion resets the
septum. 20/20 random additions correct, ~134 ms (7.5 Hz) per stage.

**`septum_viz.py`** → **`septum_paced.png`** — the generated theta rhythm with
each carry hop marked on a theta peak, plus the coherence trace dipping at each
feedback reset (the loop visibly live).

**`septumpacemakernode.py`** — the septum as a PerceptionLab node. Outputs
`theta`, `peak` (one tick per cycle), `phase`, `coherence`. Wire `peak` into a
resonator neuron's `reset` ports to pace it; wire the neuron's output into
`reset_in` to close the loop.

**`geometric_neuron_v19.json`** — the graph: the Septum pacing a v18 neuron
(soma → axon → synapse), with each theta peak re-arming the neuron and the
neuron's spike feeding back to reset the septum. Drop the v18 node files
(`resonatorsomanode.py`, `resonatoraxonnode.py`, `synapselatchnode.py`, included
here) and this node in your `nodes/` folder, then load the graph.

(Supporting: `resonator_neuron.py`, `gates.py`, `propagating_spike.py` from
earlier folders; the three v18 node files.)

## Honest scope

The arithmetic is at the verified algebraic resonator level; this folder's
contribution is the *timing* (generated, phase-resettable theta pacing the
stages), not a re-proof of the arithmetic or a full theta-paced continuous
field (that marries v18 + v19, the next build). The Kuramoto septum is the
abstract pacemaker from Vardalakis, not a biophysical septum. The
"same-mechanism-as-the-accident" claim is a structural correspondence, not an
identity. See `PAPER.md` §5.
